Introduction

Cluster analysis (or clustering) is a technique used to group data points based on their similarity. In this exercise, you will perform hierarchical clustering using Euclidean distance and Ward’s method. By the end of this tutorial, you will: - Learn how to prepare the dataset and select the respective variables, - Understand the step-by-step process of hierarchical clustering, - Learn to evaluate clustering results using dendrograms, scatterplots, elbow plots, and silhouette plots, - Understand how to report the cluster results.

Perform Hierarchical Clustering

We’ll apply Ward’s method to create clusters and plot the dendrogram.

# Perform hierarchical clustering
hc1 <- hclust(distance_matrix, method = "ward.D2")

# Display the dendrogram
plot(hc1, 
     main = "Dendrogram for Hierarchical Clustering", 
     xlab = "Observations", 
     ylab = "Height")

Interpretation of the plot: The dendrogram represents how the clusters are formed step-by-step by merging individual data points (or smaller clusters) into larger clusters. The dendrogram is like a tree, with each branch representing a data point or a cluster of data points. The further down the tree you go, the more similar the data points are to each other. The height at which two branches (clusters) join represents the dissimilarity (or distance) between them. A lower height means the clusters are very similar, while a higher height means the clusters are more different. By drawing a horizontal line across the tree at a certain height (cutting through the branches), you can decide how many clusters you want.

Interpretation of plots:

The Elbow Method visualizes the total within-cluster sum of squares (WSS) as a function of the number of clusters. The goal is to identify the “elbow point,” where the rate of decrease in WSS slows down. This suggests a good balance between minimizing WSS and avoiding too many clusters.Examine the plot and look for the point where the curve forms an “elbow” or levels off. This point represents the optimal number of clusters. If no clear elbow exists, use the Silhouette Method as a supplementary approach.

The Silhouette Method evaluates how well-separated the clusters are by calculating the silhouette width, a measure of how similar an object is to its own cluster compared to others. The higher the average silhouette width, the better-defined the clusters. Look for the number of clusters with the highest average silhouette width in the plot. In our case three clusters.

Cut the Dendrogram to Form Clusters and Visualize the Clusters

Decide on the number of clusters and assign cluster labels to the data (e.g., 3).

# Cut the dendrogram/tree to create 3 clusters
clusters <- cutree(hc1, k = 3)

# Add cluster membership to the original dataset
df1$Cluster <- clusters

#Visualize the clusters in the dendrogram
plot(hc1, cex = 0.6)
rect.hclust(hc1, k = 3, border = 2:5)
# Scatterplot of the first two dimensions (scaled data)
plot(cluster_df1[, 1:2], 
     col = clusters, 
     pch = 19, 
     main = "Scatterplot of Clusters",
     xlab = "First Dimension (scaled)",
     ylab = "Second Dimension (scaled)")
# Cluster plot
fviz_cluster(list(data = cluster_df1, cluster = clusters))

Interpretation of plots:

The dendrogram provides insight into the relationship between the data points and how they are grouped together based on their similarities. The rectangular borders highlight the three main clusters, showing the level at which each cluster was formed.

The scatterplot visualizes the first two principal components (or dimensions) of the scaled data, where each point represents an observation, colored according to its assigned cluster. Attention: The dimensions for the scatter plot or cluster plot are not necessarily the first two variables entered into the clustering. Instead, they typically represent the first two principal components. Basically, R performs a factor analysis to visualize the three-dimensional data.

The cluster plot is a more refined cluster plot that displays the data points in a 2D plot, with the clusters clearly marked by different colors and possibly showing the cluster centers.

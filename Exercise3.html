Aims & overview

This tutorial gives an introduction into comparing differences in the means of more than two groups with an analysis of variance (ANOVA). You will learn how to:

  • prepare the data for an ANOVA
  • check for homogeneity of variance
  • perform and interpret a one-way ANOVA
  • conduct and interpret post-hoc tests

Define independent and dependent variables

We would like to examine whether the recall of the ads differs significantly depending on the form of advertising employed. The variable “adformat” is therefore our independent variable (IV). We define this as a factor using the “factor” function. We use the “levels” and “labels” arguments to store the three factor levels with their associated labels. The value 1 is assigned to the radio label, the value 2 to the print label and the value 3 to the TV label.

Then we define the variable “recall” as a dependent variable. As with the t-test, the IV level of measurement must be nominal and the DV level should be metric. We use glimpse() to check if this is the case.

#define independent and dependent variables
#convert independent variable into factor
df$iv <- factor(df$adformat, 
                levels = c (1,2,3), 
                labels = c("Radio", "Print", "TV")) 

df$dv <- df$recall

We see that, on average, radio ads are remembered worse than print or TV ads. In addition, TV ads are remembered a little better than print advertising. However, this does not imply that these differences are also statistically significant. It could be that we only measure these differences by chance in our sample, but that in fact there are no differences at all (null hypothesis). To check this, we carry out a one-way analysis of variance below. In contrast to the t-test, we can test the means of more than two groups at the same time.

Levene’s test for checking for homogeneity of variance

As with the t-test, we must first check whether the requirement of homogeneity of variances is met. For this we use Levene’s test again, with which we check whether the variances in the three groups (radio, print, TV) differ significantly. If this test becomes significant, we must NOT assume variance homogeneity, i.e. the variances differ significantly between the groups (variance heterogeneity). To perform Levene’s test, the code is exactly the same as in Exercise 2: we retrieve the data set “df”, remove all cases with missing values, and then run the Levene test on our DV and IV.

df %>%
  drop_na(iv, dv) %>%
  leveneTest(dv~iv, data = .)
Levene's Test for Homogeneity of Variance (center = median)
      Df F value Pr(>F)  
group  2    3.84 0.0514 .
      12                 
---
Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

We are lucky! Levene’s test is not significant (F(2,12) = 3.84, p = 0.0514), because p > 0.05. Thus, we can retain the null hypothesis, assume homogeneity of variance, and proceed with the analysis of variance.

If Levene’s test gets significant, we should run the more robust Welch test again (see below), and also choose specific post-hoc tests accounting for heterogeneity of variance.

One-way analysis of variance (ANOVA)

First, we remove cases with missing values again with the “drop_na” function so that the mean values can be calculated. For the one-way analysis of variance itself, we use the “aov” function (from the standard “stats” package). With this function, an F-test is calculated, the information of which we store in a separate object. We call it “model”. With the “summary” function we display the result.

We use the F-test to check whether at least two of the group means differ significantly from one another. The hypothesis we are testing is undirected:

H1: The format of advertising has a significant influence on recall.

H0: The form of advertising has no influence on recall.

df <- drop_na(df, iv, dv)
model <- aov(dv~iv, data = df)
summary(model)
            Df Sum Sq Mean Sq F value   Pr(>F)    
iv           2    130   65.00   20.53 0.000134 ***
Residuals   12     38    3.17                     
---
Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

The output shows a table with two rows: iv and Residuals. Degrees of freedom (Df), sums of squares, mean squares (Mean Sq), F-values (F value) and p-values (Pr) are displayed. The values in the first row (IV) refer to the variance between factor levels, that is, the variance that can be explained by group membership (the IV). The values in the second line (Residuals) refer to the variance within the groups or factor levels, i.e. the variance that cannot be explained by the IV. We see that the IV sums of squares (Sum Sq) and mean squares (Mean Sq) are much larger than those of the residuals. The explained variance exceeds the unexplained variance by far.

The most important metrics for interpreting the ANOVA are the F-value and the p-value, from which we can infer whether the difference in the mean squares is significant. The F-value relates the explained and unexplained variance to each other. In our example, it is 20.53 with a p-value of 0.000134. The p-value is therefore clearly smaller than our desired significance level of p < 0.05. The three stars *** behind the p-value mean that the result is even highly significant, i.e. p < 0.001 (see legend below the table); ** mean highly significant with p < 0.01; * stands for significant with p < 0.05.

We can therefore conclude from the results that the format of advertising has a significant influence on recall, i.e. that at least two group mean values differ significantly from one another. However, we do not know which of the mean values these are.

We report the result as follows: The recall performance (measured by the number of recalled advertising messages) significantly differed according to the form of advertising, F(2,12) = 20.53, p < .001.

Welch’s F: In the case of a significant Levene test, i.e. if the assumption of homogeneity of variance is violated, we calculate F according to Welch. To do this, we use the “oneway.test()” function, which calculates the more robust Welch F-value by default. So instead of the code chunk above, we’ll use the code below.

df <- drop_na(df, iv, dv)
oneway.test(dv~iv, data = df)

    One-way analysis of means (not assuming equal variances)

data:  dv and iv
F = 37.262, num df = 2.0000, denom df = 6.4983, p-value = 0.0002752

Post-hoc testing

In order to analyze more precisely how many and which of the means differ significantly from one another, we carry out so-called post-hoc tests. The mean values are then compared with one another in pairs using t-tests. With the argument p.adj = “bonf” we choose the Bonferroni correction method.

df %>%
drop_na(iv, dv) %>%
  do(tidy(pairwise.t.test(.$dv, .$iv, p.adj = "bonferroni")))
# A tibble: 3 × 3
  group1 group2  p.value
  <chr>  <chr>     <dbl>
1 Print  Radio  0.00241 
2 TV     Radio  0.000134
3 TV     Print  0.303

In the table of results we can see that the difference between print and radio is significant (p = 0.0024 and therefore p < 0.05). The difference between TV and radio is also significant (p = 0.00013 and thus p < 0.05). The difference between TV and print, however, is not significant.

Together with the descriptive statistics calculated above, we can now summarize that radio advertising resulted in a significantly worse recall (M = 5, SD = 1.58) of the advertising message than print (M = 10, SD = 2.55) and TV advertising (M = 12, SD = 0.71). TV ads are remembered no better than print ads.

Note: In the case of heterogeneity of variance, i.e. a significant Levene test, we also choose a different post-hoc test procedure accounting for this, for example the code below executes the TamhaneT2-Test from the PMCMRplus package. It needs to run on our “model” object created above with the “aov” function (repeated here for completeness).

model <- aov(dv~iv, data = df)
tamhaneT2Test(model)

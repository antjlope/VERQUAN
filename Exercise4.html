Aims & overview

This tutorial follows up on the tutorial ANOVA I and extends it to a two-way analysis of variance (ANOVA). You will learn how to:

  • perform and interpret a two-way ANOVA
  • calculate and interpret effect sizes

Define IVs and DVs

We want to carry out a two-way ANOVA and check whether the recall of the respondents differs significantly according to the type of advertising and gender. The variables “adformat” and “gender” are therefore our IVs and must be nominally scaled. In the above output of the “glimpse()” function, we see that the two variables are encoded numerically (of type “integer” int), even though they are our grouping variables. We, thus, need to convert them into factors using the “factor” function.

Let’s recode the gender variable,first. It’s dichotomous with 0 = male and 1 = female. Since R sometimes interprets null values as missing values, we better recode this variable to replace all 0s with the value 1 and all 1s with the value 2. 1 then means male, 2 female.

#recode variable "gender"
df$gender <- Recode((df$gender), 
                '0 = 1; 1 = 2')

Now we define the IVs and DVs and convert the IVs to factors with the function “factor”. We use the “levels” and “labels” arguments to store the three factor levels with their associated labels.

  • “adformat”: Value 1 is assigned to “radio”, value 2 to “print” and value 3 to “TV”.
  • “gender”: Value 1 stands for “male”, value 2 for “female”.

Attention! If the values of the IVs are already as text, i.e., they are of type “character” (chr), then, we can simply use the “as.factor()” function.

The variable “recall” is our DV, and it’s already a metric variable as requested.

#define IVs and convert into factors
df$iv1 <- factor(df$adformat, 
                levels = c (1,2,3), 
                labels = c("Radio", "Print", "TV")) 
df$iv2 <- factor(df$gender, 
                levels = c (1,2), 
                labels = c("Male", "Female"))

First, however, we group the data according to the two IVs with the “group_by” function and use the “summarize” function to calculate summary statistics for our DV: mean values (mean = mean (dv)), standard deviations (std_dev = sd(dv)) and the number of values per group (count = n()). To link the different working steps together, we use the pipe again (%>%). We store the statistics in a new object called “df_by_group”.

Levene’s test for checking homogeneity of variance

Before we carry out the ANOVA, we have to check whether the condition of homogeneity of variance is met. To do this, we use Levene’s test again, which tests whether the variances in all six groups differ significantly from one another. The commands below are already known from the previous exercises. But since we now have two IVs, we combine them in the form: iv1*iv2

df %>%
  drop_na(iv1, iv2, dv) %>%
  leveneTest(dv ~ iv1*iv2, data = .)
Levene's Test for Homogeneity of Variance (center = median)
      Df F value Pr(>F)
group  5   1.785  0.212
       9

Levene’s test is not significant (F(5,9) = 1.78, p = 0.212) because p > 0.05. Thus, we can retain the null hypothesis, assume homogeneity of variance, and proceed with the analysis of variance.

Two-way ANOVA

For conducting the ANOVA, we use the “aov” function again. Also in this function we enter the two IVs linked with *. This way, we make sure that both factors are tested individually for their respective main effects as well as in combination, i.e., as an interaction effect. If you want to check the two factors only separately, without calculating an interaction effect, we use a +.

df <- drop_na(df, iv1, iv2, dv)
model <- aov(dv ~ iv1*iv2, data = df)
summary(model)
            Df Sum Sq Mean Sq F value   Pr(>F)    
iv1          2 130.00   65.00  20.893 0.000415 ***
iv2          1   6.67    6.67   2.143 0.177268    
iv1:iv2      2   3.33    1.67   0.536 0.602811    
Residuals    9  28.00    3.11                     
---
Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

A table with four rows is displayed in the output. The columns correspond exactly to those of the one-way ANOVA (see Exercise 3). The line “iv1” contains the statistics for the main effect of our first IV (adformat), the line “iv2” contains the statistics for the main effect of our second IV (gender). We see in the last column that the main effect for IV1 is highly significant with p < 0.001. The main effect for IV2, in contrast, is not significant (p=0.18 and thus > 0.05).

In the third line, we see the interaction effect. This is also not significant in our example, i.e. gender and form of advertising do not have any specific effects in certain combinations of the two variables.

The values in the fourth row (residuals) refer to the within-group variance that cannot be explained by the IVs and/or their interaction effect (sometimes called error variance).

Effect sizes

In order to determine exactly how large the influence of the IVs on the DV is and to be able to compare the size of the various significant (!) effects with one another, we determine the effect sizes.

eta_squared(model)
# Effect Size for ANOVA (Type I)

Parameter | Eta2 (partial) |       95% CI
-----------------------------------------
iv1       |           0.82 | [0.54, 1.00]
iv2       |           0.19 | [0.00, 1.00]
iv1:iv2   |           0.11 | [0.00, 1.00]

- One-sided CIs: upper bound fixed at [1.00].

Important! We only interpret the effect sizes of the significant effects, in this case, for the main effect of IV1. The eta squared is 0.82. According to Cohen, this is a large effect (small = 0.01; medium = 0.06; large = 0.14).

Note: The partial eta squares are calculated here using the Type1 method. This means that the effect sizes account for the variance explained by the previous variables in the model, i.e. the partial eta square of IV2 measures the proportion of variance explained by IV2 of the total variance remaining after accounting for the variance already explained by IV1, and so forth.

Report results

We report the results of the two-way ANOVA as follows:

The main effect of the advertising format on recall is significant. The number of recalled advertising messages differs statistically significantly for the different advertising formats, F (2.9) = 20.89, p < 0.001, η² = 0.82. The main effect of gender (p = 0.18) and the interaction effect of advertising format and gender (p = 0.60) are not significant.

If we hadn’t already done this in Exercise 3, we would now proceed with the post-hoc tests for the significant main effect to find out which of the groups are significantly different from each other.

Visualization of the effects

We now create a line chart with error bars to visualize the main effect of our IV1 (adformat). First, we create a variable called “Title” and assign to it the title we want to give the figure: “Effect of Advertising Format”

Then, we use the “ggline” function from the “ggpubr” package to create a line chart. We take our data set “df” and, first, remove all rows with missing values for our IV1 and DV with “drop_na()”. Then, we specify which variables to use for the x-axis and the y-axis of the chart. We put the IV1 on the x-axis and the DV on the y-axis.

With the argument “title = Title” we specify the title chosen above as the title for the graph and add the 95% confidence intervals as error bars with “add” (mean_ci). Finally, we use “xlab” and “ylab” to set the labels for the axes of the chart, referencing our labels we created above (see the “Descriptive Statistics” code chunk).

Title <- "Effect of Advertising Format"

ggline(df %>% drop_na (dv, iv1),
        x = "iv1", 
        y = "dv",
        title = Title,
        add = "mean_ci",
        xlab = label_iv1,
        ylab = label_dv)

As a result, we get a line chart showing the mean values of the recall variable (DV) for the different formats of advertising (IV1). The confidence intervals indicate the range of values in which the true mean values lie with 95% certainty. These vary in size for the various formats of advertising, and they partially overlap, particularly for print and TV. This suggests that the differences in the means of these two formats of advertising are not significant.

An interaction diagram is suitable for visualizing both main effects and their potential interaction. We create such a diagram with the “interaction.plot” function from the “stats” package. We define the following arguments:

  • df$iv1: The values of IV1 are displayed on the x-axis.
  • df$iv2: The values of IV2 are used to create different lines in the plot.
  • df$dv: The values of our DV are displayed on the y-axis.
  • fun = mean: The “mean” function is used to calculate the mean values of DV for each factor level of IV1 and IV2. These are used to create the lines in the plot.
  • ylab = label_iv1: The label for the y-axis is set to “label_iv1” which we set above.
  • xlab = label_dv: The label for the x-axis is set to “label_dv”, which we also defined above.
  • trace.label = label_iv2: The legend for the lines in the plot is labeled with “label_iv2”.
  • col = c(“#0198f9”, “#f95801”): The colors for the lines are set to blue and orange.
  • lty = 1: The line type for the lines is set as a solid line (lty=1).
  • lwd = 2: The thickness of the lines is set to 2.
  • xtick = TRUE: Grid lines for the x-axis are displayed.
interaction.plot(
df$iv1, df$iv2, df$dv,
fun = mean,
ylab = label_dv,
xlab = label_iv1,
trace.label = label_iv2,
col = c("#0198f9", "#f95801"),
lty = 1,
lwd = 1,
xtick = T,
)

The plot shows what we could already read from the tables of the mean values. While there are gender differences in the mean values for radio and print ads, they disappear for TV ads. However, from our ANOVA we already know that neither the main effect for gender nor the interaction effect of our two IVs are significant. We therefore do not further interpret the interaction diagram at this point.

Aims & overview

This tutorial teaches you how to compare differences in the means between two groups using a t-test. You will learn how to:

  • prepare the data for a t-test
  • test for the assumption of variance homogeneity using the Levene test
  • calculate, interpret and report results of a t-test.

Define independent and dependent variables

We would like to check whether the average ratings of “comedy” or “drama” differ significantly. To do this, we first use “filter()” to select the cases in which the genre corresponds to either “Comedy” or “Drama”. We save this reduced data set as a new object of class “dataframe” named “df”.

df <- filter(netflix_films, Genre == "Comedy" | Genre == "Drama" )

Then, we define the variable “Genre” as the independent variable (iv) and the variable “IMDB.Score” as the dependent variable (dv). An important prerequisite for the t-test is that the scale level of the IV is nominal and dichotomous (i.e. it has only two potential values).Therefore, we save the new variable “iv” as a factor with the command “as.factor”.

#Define independent and dependent variables
df$iv <- as.factor(df$Genre)#we convert the IV into a factor
df$dv <- df$IMDB.Score

We use “View(df)” to look at the data frame and check whether filtering the cases and creating the new variables worked the way we wanted. We use “glimpse()” to check whether “iv” is saved as a factor and “dv” is a numeric variable.

We see that films of genre “drama” are rated higher on average than “comedy” films. However, this does not necessarily mean that this difference is also statistically significant. It could be that we only measure this difference in our sample by chance, but in fact, that there is no difference at all. To check this, we perform a t-test for independent samples below.

Levene’s test for checking the assumption of variance homogeneity

Before we do the t-test itself, we have to check an important prerequisite: variance homogeneity! We want to know whether the variances in the two groups (drama vs. comedy) are significantly different. To do this, we perform Levene’s test. Depending on this result, we choose the further test procedure.

As a first step, we call the “df” data set. Since we want to connect several data handling steps in a sequence, we work with the %>% operator. It is called the “pipe”, and you can read it as “and then”. We take our data set “df”, and then we remove all missing values (cases where our iv or dv are missing) using the “drop_na()” function from the tidyverse package; and then we do Levene’s test.

With the argument dv ~ iv we define the variables for which homogeneity of variance will be tested.

Since we defined the data set we work with before we started the pipe, we do not have to spell it out subsequently, but can just use the argument “data = .” to refer to it.

Question: What are H0 and H1 for Levene’s test?

df %>%
  drop_na(iv, dv) %>%
  leveneTest(dv~iv, data = .)
Levene's Test for Homogeneity of Variance (center = median)
       Df F value Pr(>F)
group   1   0.078 0.7805
      124

Levene’s test is not significant (F(1,124) = 0.08, p = 0.78) because p is greater than 0.05. Thus, the null hypothesis is retained: we can assume variance homogeneity, i.e. that the variances of both genre groups are equal, and continue with the t-test.

Important! If Levene’s test was significant, i.e. the variances of the values in the two groups were unequal, we would need to use the Welch test instead of the t-test for independent samples, which does not require variance homogeneity (see below).

t-test

We now perform the t-test. The logic and structure of the code is the same than for Levene’s test, except that we now use the command “t.test”. We use the same dv and iv as before and test “two.sided” for the significance of the mean differences in both directions (drama > comedy and drama < comedy). This means that the hypothesis we are testing is undirected:

H1: Drama films are rated significantly differently than comedy films. H0: There is no difference in the ratings of drama and comedy films.

The argument “var.equal = T” is used if Levene’s test is not significant and, thus, there is homogeneity of variance. If this is not the case, we must set “var.equal = F”. Then, the Welch test instead of the t-test is calculated.

df %>%
  drop_na(iv, dv) %>%
  t.test(dv~iv, 
         alternative = "two.sided",
         var.equal = T,
         data = .)

    Two Sample t-test

data:  dv by iv
t = -5.9272, df = 124, p-value = 2.846e-08
alternative hypothesis: true difference in means between group Comedy and group Drama is not equal to 0
95 percent confidence interval:
 -1.116146 -0.557323
sample estimates:
mean in group Comedy  mean in group Drama 
            5.506122             6.342857

In the output, the first line shows for which dv and iv the t-test was calculated, followed by the test statistics in the second line (t = -5.9272, df=124, p-value=2.846e-08). The t-value is -5.9272 with 124 degrees of freedom. The negative sign indicates that the comedy group mean is less than the drama group mean. The p-value is p=2.846e-08 or p=0.00000002846. This is significantly smaller than 0.05 and even significantly smaller than 0.001, i.e. p < 0.001 and therefore highly significant.

We report the result as follows: Comedy films (M = 5.51, SD = 0.79) are rated significantly worse than drama films (M = 6.34, SD = 0.76, t = -5.9272, df=124, p<0.001).
